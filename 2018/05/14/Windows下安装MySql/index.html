Windows下安装MySql

发表于 2018-05-14 更新于 2018-05-15 分类于 数据库 阅读次数：   |     Valine：
本文字数： 740 阅读时长 ≈ 1 分钟

下载MySql

  1. 下载地址：https://dev.mysql.com/downloads/mysql/
  2. 下载zip包：
  3. 下载后解压：D:\软件安装包\mysql-5.7.20-winx64

环境变量配置

  • 变量名：MYSQL_HOME
  • 变量值：D:\软件安装包\mysql-5.7.20-winx64
  • path里添加：%MYSQL_HOME%\bin;

    生成data文件：

以管理员身份运行cmd,进入D:\软件安装包\mysql-5.7.20-winx64\bin下，执行命令：mysqld --initialize-insecure --user=mysql

启动服务

执行命令：net start mysql启动mysql服务，若提示：服务名无效等问题详见问题解决，若成功启动服务则安装成功。

问题解决

提示：服务名无效
解决方法：
执行命令：mysqld -install即可
若提示服务正在启动或停止中，请稍候片刻再试一次。则需要先去资源管理器中把mysql进程结束，重新启动即可。

登录mysql

以管理员身份运行cmd,进入D:\软件安装包\mysql-5.7.20-winx64\bin下，执行命令：mysql -u root -p
因为之前没设置密码，所以密码为空，不用输入密码，直接回车即可。

设置或修改密码

  1. 修改
update mysql.user set authentication_string=password("123456") where user
="root"

注意password(“123456”),此处引号中的内容是密码，自己可以随便设置

  1. 保存

flush privileges
作用：相当于保存，执行此命令后，设置才生效，若不执行，还是之前的密码不变
